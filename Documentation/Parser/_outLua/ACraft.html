Abstract Class:

ACraft

A flying Actor which carries other things and can drop them.

Parent: Actor


Property:

ClassName

The class name of this Entity. A string with the friendly-formatted type name of this object.


Function:

OpenHatch

Opens the hatch doors, if they're closed or closing.

Arguments:
None.

Return value:
None.


Function:

CloseHatch

Closes the hatch doors, if they're open or opening.

Arguments:
None.

Return value:
None.


Read-Only Property:

HatchState

The current state of the hatch. An int encoding the hatch state. See the HatchState enum.


Read-Only Property:

MaxPassengers

The recomended, not absolute, maximum number of actors that fit in the invetory. Used by the activity AI. An integer with the recomended number of actors that fit in the craft. Default is -1 (unknown).


Read-Only Property:

DeliveryDelayMultiplier

Delivery delay multiplier.
