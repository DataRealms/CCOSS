Concrete Class:

ADoor

A sliding or swinging door.

Parent: Actor


Property:

ClassName

The class name of this Entity. A string with the friendly-formatted type name of this object.


Read-Only Property:

Door

The moving door Attachable A pointer to the door Attachable of this. Ownership is NOT transferred!


Function:

GetDoorState

Gets the current state of the door.

Arguments:
None.

Return value:
An int encoding the hatch state. See the DoorState enum.


Function:

OpenDoor

Opens the door, if it's closed.

Arguments:
None.

Return value:
None.


Function:

CloseDoor

Closes the door, if it's open.

Arguments:
None.

Return value:
None.


Function:

SetClosedByDefault

Sets whether this ADoor closes after a while by default (or opens)

Arguments:
Whether the door by default goes to a closed position. If not, then it will open after a while.

Return value:
None.
